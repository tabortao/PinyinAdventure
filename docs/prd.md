# 智能拼音大闯关游戏需求文档

## 1. 游戏概述

### 1.1 游戏名称
智能拼音大闯关

### 1.2 游戏描述
基于艾宾浩斯记忆曲线，使用AI驱动的拼音学习闯关游戏，帮助用户高效学习和巩固汉语拼音知识。

### 1.3 玩法说明
- 游戏采用闯关模式，按照难度等级划分关卡
- 用户点击「开始Level X」进入游戏页面
- AI驱动系统实时搜索对应年级的人教版生字表内容，动态生成拼音题目
- 用户使用带有音调的键盘输入对应拼音
- 回答正确显示英文庆祝语，回答错误记录到「错音表」
- 系统基于艾宾浩斯记忆曲线，智能安排复习内容
- 积分奖励系统：每关10道题，答对1题得1分，闯关过程中实时显示得分

## 2. 功能模块

### 2.1 闯关系统
- 关卡划分：Level 1到Level 6，每个Level包含多个关卡，最后为AI复习关
- 题目来源：AI实时搜索人教版一年级到六年级生字表内容
- 题目类型：单个汉字、词语、句子
- 答题方式：拼音键盘输入，支持声调选择
- 积分规则：每关10道题，答对1题得1分，重新闯相同关分数不累加

### 2.2 错音表功能
- 记录用户答错的拼音内容
- 按日期统计错误情况
- 艾宾浩斯曲线智能复习提醒
- AI复习关重点练习错音表中的内容

### 2.3 模式设置
- 汉字模式：单个汉字拼音练习
- 词语模式：词语拼音练习，最长10个字
- 句子模式：句子拼音练习，最长10个字
- 设置入口：首页右上角设置按钮

### 2.4 交互反馈
- 正确回答：英文庆祝语鼓励，钢琴按键音效
- 错误回答：正确答案提示
- 进度保存：自动记录闯关进度
- 字体调整：句子模式下题目框字体自动缩小，优化排版美观性

### 2.5 积分系统
- 游戏过程中实时显示当前得分
- 首页展示总积分（所有关卡得分汇总）
- 积分排行榜功能

## 3. 功能组件

### 3.1 拼音键盘
- 标准拼音输入键盘，带声调选择
- 增加空格键，支持词语和句子模式中的字间距
- 钢琴按键音效反馈

### 3.2 题目显示区域
- 智能字体适配：根据题目长度自动调整字体大小
- 清晰的题目边框和背景区分
- 支持多种题目格式显示

## 4. 数据基础

### 4.1 生字表搜索
- 技术实现：AI搜索引擎实时抓取人教版各年级生字表内容
- 动态生成：根据年级难度动态生成对应拼音题目
- 避免重复：智能去重，确保题目多样性
- 自适应：根据不同模式自动调整题目形式

### 4.2 艾宾浩斯算法
- 智能安排复习时间点
- 基于遗忘曲线的强化记忆
- 个性化学习节奏调整

## 5. 设计风格

### 5.1 配色方案
- 主色调：清新的蓝绿色系，营造学习氛围
- 辅助色：温暖的橙黄色用于正确反馈，柔和的粉色用于提示信息

### 5.2 视觉细节
- 圆角设计：按钮和卡片采用8px圆角，增加亲和力
- 卡通风格：界面元素采用可爱卡通造型，吸引儿童用户
- 动画效果：答题正确时添加庆祝动画效果

### 5.3 整体布局
- 卡片式布局：闯关按钮采用网格排列
- 清晰分区：游戏区域、键盘区域、功能按钮区域明确分离
- 积分显示区域：首页和游戏过程中醒目的积分展示位置
