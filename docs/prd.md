# 智能拼音大闯关游戏需求文档

## 1. 游戏概述

### 1.1 游戏名称
智能拼音大闯关

### 1.2 游戏描述
基于艾宾浩斯记忆曲线，使用AI驱动的拼音学习闯关游戏，帮助用户高效学习和巩固汉语拼音知识。

### 1.3 玩法说明
- 游戏采用闯关模式，按照难度等级划分关卡
- 用户点击「开始Level X」进入游戏页面
- AI驱动系统实时搜索对应年级的人教版生字表内容，动态生成拼音题目
- 用户使用带有音调的键盘输入对应拼音
- 回答正确显示英文庆祝语，回答错误记录到「错音表」
- 系统基于艾宾浩斯记忆曲线，智能安排复习内容
- 积分奖励系统：每关10道题，答对1题得1分，闯关过程中实时显示得分

## 2. 功能模块

### 2.1 闯关系统
- 关卡划分：Level 1到Level 6，每个Level包含多个关卡，最后为AI复习关
- 题目数量：每个关卡固定10道题，通过增加关卡数量来丰富内容
- 题目来源：AI实时搜索人教版一年级到六年级生字表内容
- 题目类型：单个汉字、词语、句子
- 答题方式：拼音键盘输入，支持声调选择
- 积分规则：每关10道题，答对1题得1分，重新闯相同关分数不累加

### 2.2 Level 1基础拼音关卡
- 声母闯关：b p m f d t n l g k h j q x zh ch sh r z c s y w
- 韵母闯关：单韵母、复韵母、前鼻韵母、后鼻韵母、整体认读音节
- 基础拼读：常见声母与韵母组合练习
- 渐进难度：从单一拼音识别到简单拼读组合

### 2.3 错音表功能
- 记录用户答错的拼音内容
- 按日期统计错误情况
- 艾宾浩斯曲线智能复习提醒
- AI复习关重点练习错音表中的内容

### 2.4 模式设置
- 汉字模式：单个汉字拼音练习
- 词语模式：词语拼音练习，最长10个字
- 句子模式：句子拼音练习，最长10个字
- 设置入口：首页右上角设置按钮

### 2.5 交互反馈
- 正确回答：连续答对给出不同英文庆祝语和动画效果
- 错误回答：汉语鼓励话语和正确答案提示
- 进度保存：自动记录闯关进度
- 字体调整：句子模式下题目框字体自动缩小，优化排版美观性

### 2.6 响应式设计
- 适配手机、平板等不同设备屏幕大小
- 自动调整界面布局和字体大小
- 保证在各种设备上的良好用户体验
- 界面元素位置自动适配

### 2.7 筛选功能
- 首页添加筛选图标
- 点击筛选图标显示不同Level的内容选择
- 支持快速切换各个学习级别
- 界面简洁明了便于用户操作

### 2.8 移动端优化
- 手机浏览器模式下Level筛选与标题「拼音闯关大冒险」避免重叠
- 优化移动端触摸区域和按钮尺寸
- 确保移动端操作流畅性

### 2.9 积分系统
- 游戏过程中实时显示当前得分
- 首页展示总积分（所有关卡得分汇总）
- 积分排行榜功能

### 2.10 帮助与说明
- 图形化设计的帮助页面
- 友好的操作指引和功能说明
- 丰富的视觉元素提升用户体验

## 3. 学习页面功能

### 3.1 学习页面结构
- 左侧导航：「学习」「闯关」两个主要页面
- 「学习」页面按关卡进行拼音学习
- 「闯关」页面为原闯关地图功能

### 3.2 拼音学习闯关
- 关卡分类：按声母、韵母、前鼻音韵母、后鼻音韵母、整体音节逐步进行闯关
- 拼音卡片内容：
  - 拼音显示
  - 帮记话语
  - 助记emoji图标
  - 发音图标（点击发音）
  - 对应词语（词语汉字上方标注拼音）

### 3.3 拼音学习测试
- 打乱拼音顺序让用户选择是否记住
- 实时反馈学习效果
- 增强用户学习参与感

### 3.4 拼音库管理
- 记录每个拼音的学习次数
- 标记是否记住状态
- 数据统计和跟踪

### 3.5 智能复习功能
- 基于拼音库中未记住的拼音
- AI驱动艾宾浩斯记忆曲线算法
- 定制化复习计划
- 个性化学习推荐

### 3.6 看字识音闯关
- 位置：学习页面下方新增「看字识音」按钮
- 页面结构：
  - 左上角返回按钮
  - 每关10道题，每题10秒答题时间
  - 总时长100秒，满分10分
  - 系统显示汉字，下方给出4个易混淆拼音选项
  - 用户点击选择正确拼音
  - 答对得1分，答错记录到「错音本」
  - 每关结束后得分计入顶部导航栏总分数
  - 完成一关后可进入下一关
- 特点：
  - 排版风格与「闯关」页面一致
  - 使用相同汉字数据源
  - 每答对一题播放对应音效
